# Muhammad Solaiman Portfolio

Static personal portfolio website for GitHub Pages.

## How to publish on GitHub Pages

1. Create a new GitHub repository named `portfolio` or `muhammad-solaiman`.
2. Upload these files:
   - index.html
   - style.css
   - script.js
   - assets folder
3. Go to Repository Settings → Pages.
4. Select Branch: `main` and Folder: `/root`.
5. Save.
6. Your website will be live in a few minutes.

## Edit contact links

Open `index.html` and update:
- Email
- GitHub
- LinkedIn
- Fiverr profile link
- Project links
